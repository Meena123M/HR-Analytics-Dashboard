# HR Analytics Dashboard (Power BI)

## Overview

The HR Analytics Dashboard is an interactive Power BI project designed to help organizations analyze workforce data and gain actionable HR insights. The dashboard provides key performance indicators (KPIs), employee demographics, attrition analysis, salary distribution, and department-wise performance to support data-driven decision-making.

---

## Features

- Employee Overview
- Active Employee Count
- Employee Attrition Analysis
- Attrition Rate KPI
- Average Employee Age
- Average Work Experience
- Department-wise Employee Distribution
- Attrition by Department
- Attrition by Salary Slab
- Attrition by Gender
- Age Group Distribution
- Attrition Trend by Experience
- Job Role & Job Satisfaction Analysis
- Interactive Age Group and Department Filters
